# Retail Sales Dashboard using Power BI

## Project Overview
This project analyzes retail sales data and provides business insights through an interactive Power BI dashboard.

## Key Metrics
- Total Sales
- Total Profit
- Total Orders

## Visualizations
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Region Filter

## Tools Used
- Power BI
- CSV Dataset
- Data Visualization
